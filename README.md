# ChamAsesor

ChamAsesor es una maqueta web para gestionar cuadrantes, turnos, plateria y tareas de cierre.

## Estado actual

- Pantalla de entrada con contraseña de responsable.
- Carga de Excel general semanal.
- Opcion para foto de planilla hecha a mano.
- Creacion de planilla de plateria desde ayudantes.
- Vista previa de rangos y tareas de cierre.
- Tareas de cierre editables.
- Panel "Antes de imprimir".

## Abrir

Abrir `index.html` en el navegador.

Contraseña inicial:

```text
chamasesor2026
```

## Pendiente

- Lectura real del Excel.
- Motor automatico de reglas.
- Generacion real de PDF y PNG.
